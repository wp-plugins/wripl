== Description ==

Wripl is a service which provides personalised, user controlled recommendations across the web.

Wripl is a hub for a user's interests, where they can manage and control their interests. Websites can then tap into wripl to provide the user with recommendations to content of interest.

== Installation ==

1. Upload `wripl-wordpress-plugin` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Contact wripl for api tokens, and fill them out under the wripl settings.
